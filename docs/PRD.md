# 📝 마이링크 (MyLink) - PRD (제품 요구사항 정의서)

## 1. 프로젝트 개요
- **프로젝트명:** 마이링크 (MyLink)
- **목적:** 흩어져 있는 다양한 소셜 미디어, 블로그, 포트폴리오, 쇼핑몰 등의 링크를 하나의 단일 URL로 관리할 수 있는 서비스입니다.
- **서비스 정책:** 모든 기능이 제한 없이 평생 100% 무료로 제공됩니다.
- **대상 사용자:** 
  - 다수의 채널을 운영하며 관리가 필요한 **크리에이터 / 인플루언서**
  - 포트폴리오 링크를 모아야 하는 **프리랜서 / 취업준비생**
  - 매장 위치, SNS, 예약 링크 등을 통합하고 싶은 **소상공인**

---

## 2. 핵심 기능 목록

### 🟢 필수 기능 (Must-Have)
> 앱의 가장 뼈대가 되는 핵심 기능들로, MVP(Minimum Viable Product) 출시에 필수적인 항목들입니다.

1. **사용자 계정 관리 (User Auth)**
   - 구글 소셜 로그인(Google OAuth)을 통한 간편 회원가입 및 로그인 (이메일 가입 미지원)
2. **프로필 관리 (Profile Management)**
   - 사용자별 고유 URL 제공 (예: `mylink.com/username` - @ 기호 없음)
   - 닉네임 및 짧은 소개글 작성
3. **링크 관리 시스템 (Link Management)**
   - 타이틀과 URL이 포함된 링크 추가, 수정, 삭제 기능
   - 사용자가 입력한 URL을 기반으로 구글 파비콘 API(Google Favicon API)를 활용해 각 웹사이트의 아이콘을 자동으로 가져와 표시

### 🟡 추후 구현 예정 기능 (Future Scope / Phase 2)
> MVP 버전 이후에 추가될 부가 기능입니다. (이 기능 또한 추후 무료로 제공됩니다.)

1. **[트래픽 및 방문자 통계 분석 (Analytics)]**
   - 누적 방문자 수 (Page Views)
   - 각 링크별 개별 클릭 수 및 통계

## 3. 기능의 상세 설명

### 3.1 사용자 계정 및 URL 생성
* 사용자가 가입 시, 본인이 원하는 영문/숫자 조합의 고유 ID를 입력합니다. 이 ID는 곧 사용자의 접속 주소(`mylink.com/username`)가 됩니다. 중복된 ID 생성은 방지해야 합니다.

### 3.2 대시보드 (관리자 뷰)
* 로그인 직후 마주하는 화면으로, 프로필 닉네임과 소개글을 수정하고 링크를 관리하는 컨트롤 패널이 제공됩니다.

### 3.3 링크 편집기 (Link Editor)
* 사용자는 [링크 추가] 기능을 통해 타이틀과 목적지 URL을 입력하여 항목을 생성합니다. 이때 등록한 URL의 웹사이트 파비콘(Favicon)을 구글 API를 통해 자동으로 추출 및 표시해 주며, 등록된 링크들의 정보를 자유롭게 수정하거나 삭제할 수 있습니다.

### 3.4 접속자 화면 (퍼블릭 뷰)
* 방문자에게 공유되는 페이지(`mylink.com/username`)로, 소유자가 등록한 프로필(닉네임, 소개)과 링크 목록만이 심플하게 표시됩니다. 방문자가 링크 항목을 클릭하면 해당 웹사이트가 **새 창(새 탭)**으로 열립니다.

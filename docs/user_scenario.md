# 🗺️ 마이링크 (MyLink) - 사용자 시나리오 (User Scenario)

## 1. 방문자: 마이링크 페이지 방문 과정

- **접속:** 방문자는 소유자의 링크 모음을 확인하기 위해 `mylink.com/username` 주소로 접속한다.
- **조회:** 방문자는 소유자가 공유하려는 웹사이트 목록을 확인하기 위해 화면에 나열된 링크 버튼(파비콘 및 타이틀)들을 살핀다.
- **이동:** 방문자는 관심 있는 콘텐츠나 웹사이트로 넘어가기 위해 원하는 링크 버튼을 클릭하여 새 창(새 탭)으로 대상 페이지를 띄운다.

---

## 2. 소유자: 링크 추가/수정/삭제 과정

### 2.1 링크 추가 (Create)
- **대시보드 진입:** 소유자는 내 페이지의 링크들을 관리하기 위해 구글 계정으로 로그인하여 관리자 대시보드에 접속한다.
- **생성 시작:** 소유자는 새로운 바로가기 항목을 만들기 위해 [링크 추가] 버튼을 누른다.
- **정보 입력:** 소유자는 방문자가 클릭할 목적지를 지정하기 위해 '타이틀'과 'URL'을 입력한다.
- **저장 및 반영:** 소유자는 새 항목을 실제 페이지에 반영하기 위해 입력을 마치고 '저장'을 누른다. (이때 시스템이 URL을 기반으로 구글 파비콘 API에서 아이콘을 자동으로 불러와 적용한다.)

### 2.2 링크 수정 (Update)
- **수정 진입:** 소유자는 이미 등록된 대상의 주소나 이름이 변경됨을 알리기 위해 기존 링크 항목에 있는 [수정] 버튼을 누른다.
- **정보 변경:** 소유자는 최신 정보로 갱신하기 위해 기존의 '타이틀' 또는 'URL'을 지우고 새로운 내용을 입력한다.
- **수정 완료:** 소유자는 변경된 데이터를 방문자 페이지에 적용하기 위해 '수정 완료(또는 저장)' 버튼을 누른다.

### 2.3 링크 삭제 (Delete)
- **삭제 진입:** 소유자는 더 이상 사용하지 않거나 방문자에게 숨기고 싶은 링크를 제거하기 위해 해당 항목의 [삭제] 버튼을 누른다.
- **삭제 확정:** 소유자는 실수로 삭제하는 것을 방지하기 위해 나타나는 "정말 삭제하시겠습니까?" 팝업 알림에서 [확인]을 눌러 해당 링크를 내 목록에서 영구적으로 없앤다.
